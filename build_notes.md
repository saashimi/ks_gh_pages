12/20/21 build notes

### Prerequisites
1. Install Homebrew
	`ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"`
2. Install Node
	`brew install node`
3. Install gulp-cli
	`npm install --global gulp-cli`

### Run
1. run `gulp` to build/minify.
2. `npm run` to run and build live preview.
